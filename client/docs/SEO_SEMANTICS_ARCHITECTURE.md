## 2. SEO Architecture (Page Structure)

```
/
├── / (Home)
├── /menu
├── /about
└── /contact
```

### Hierarchy
- **Home** — Hub, main landing, primary keywords
- **Menu** — Product catalog, pizza names, formules
- **About** — Trust, story, USP
- **Contact** — Local, hours, map, CTA

### Internal Linking
- Home → Menu, About, Contact
- Menu → Contact (order CTA)
- About → Menu, Contact
- Contact → Menu
